# SafeNet – Cyber Safety App

SafeNet is a Flutter-based mobile application designed to spread awareness about digital arrest scams and help users stay safe online.

## Features

* Multi-screen navigation (Home, Tips, Helpline, Report)
* Awareness about digital arrest scams
* Preventive cybersecurity tips
* Cybercrime helpline integration

## Tech Stack

* Flutter (Dart)

This is a simplified version of the project. Full assets and original structure are not included.
